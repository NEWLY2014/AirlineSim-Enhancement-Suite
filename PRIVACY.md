# Privacy Policy

Last updated: 2026-04-10

AirlineSim Enhancement Suite ("AES") is a browser extension for improving the AirlineSim gameplay interface.

## Data Collection

AES does not collect, sell, share, or transmit personal data to the developer or to third parties.

The extension reads information from AirlineSim pages where it is installed and stores derived gameplay data locally in the user's browser by using Chrome extension storage. This may include airline settings, route and schedule data, inventory pricing history, personnel management status, competitor monitoring data, and aircraft profitability data.

## Local Storage

AES stores data only on the user's device through `chrome.storage.local`. This data is used to provide extension features such as dashboards, history comparison, filtering, backup and restore, and reminders inside AirlineSim pages.

Users can remove stored AES data from the extension options page.

## Backup and Restore

If a user chooses to create a backup, AES exports selected local extension data into a file controlled by the user. If a user restores a backup, AES imports the selected file back into local extension storage.

## Network Access

AES runs only on `https://*.airlinesim.aero/*` pages. It does not send stored data to external servers. Links in the AES menu may open external support or documentation pages only when the user clicks them.
